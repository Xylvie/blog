## ** WebBlog Platform **

A blogging platform with an admin dashboard that allows post creation, category management, and comment moderation. Supports anonymous comments and lets post owners control their content.

## Features
- Admin dashboard to create and manage posts and categories
- Automatic dedicated pages for each post
- Anonymous comments for non-logged-in users
- Post filtering by category
- Only owners can edit/delete their posts and comments

## Tech Stack
- Laravel
- TailwindCSS
- JavaScript

## Usage
Access /admin to manage posts and categories

Public users can view posts and comment anonymously
